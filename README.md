## Requiments

- nvim 0.11
- tmux
- wezterm

## Getting started
- Give permissions to `script.sh`
```
chmod +x script.sh
```

- Run script for automatic setup:
```
./script.sh
```

- Setup env secret:
Copy .env.sample to ~/.env and setup env variables
```
cp .env.sample ~/.env
vi ~/.env
```
